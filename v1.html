Grfico de reas

Este es un tipo de grfico, similar a uno de lneas, en el que el rea entre la lnea y el eje x se pinta de un color dado, de manera que pueda aportar ms informacin a la presentacin de los datos. Para este caso, se va a intentar responder una pregunta sencilla, Se est perdiendo poblacin en la zona pirenaica navarra?

Se cargan las principales librerias que se van a emplear para esta visualizacin

  • Matplotlib: para la visualizacin de datos

Grfico de rea de 2 colores

La ventaja de usar un grfico de rea es que nos permite rellenar con 2 colores y visualizar rpidamente zonas. En este ejemplo, en lugar de mostrar la poblacin, se va a analizar la variacin de poblacin de un ao a otro.

In[6]:
# Ordenar los indices anualmetne
total_pirineo = total_pirineo.sort_index()

# Aseguramos los valores numericos
total_pirineo = total_pirineo.astype(float)

# Calculamos la diferencia de poblacion entre cada ao
delta = total_pirineo.diff()
delta = delta.dropna()

x = delta.index.astype(int)    
y = delta.values              

fig, ax = plt.subplots(figsize=(10, 4))

# rea positiva
ax.fill_between(
    x, y, 0,
    where=(y >= 0),
    interpolate=True,
    color="#2ca02c",   # verde
    alpha=0.6,
    label="Positiva"
)

# rea negativa
ax.fill_between(
    x, y, 0,
    where=(y < 0),
    interpolate=True,
    color="#d82929",   # rojo
    alpha=0.6,
    label="Negativa"
)

ax.axhline(0, color="black", linewidth=1)
ax.set_title("Pirineo  variacin anual de poblacin")
ax.set_xlabel("Ao")
ax.set_ylabel("Variacin habitantes vs. ao anterior")
ax.legend()
ax.set_xticks(x)
plt.tight_layout()
plt.show()

Regresin lineal

Para tener un resultado ms claro a la hora de proponer una respuesta, se puede hacer una regresin lineal con los datos para tener una lnea de tendencia. Dependiendo de la pendiente de dicha recta, se podr determinar ms fcil y se complementar con el grafico de rea si la poblacin aumenta o disminuye.

In[7]:

coef = np.polyfit(x, y, 1)
a, b = coef
tendencia = a * x + b

Respuesta:

Se est perdiendo poblacin en la zona pirenaica navarra?

En el grfico se representan las variaciones anuales de poblacin del conjunto de municipios de la zona pirenaica de Navarra. Las reas en verde indican aos en los que la poblacin total aument respecto al ao anterior, mientras que las reas en rojo sealan aos de prdida de habitantes. Esta visualizacin es til porque permite identificar de un vistazo no solo la direccin del cambio demogrfico, sino tambin su intensidad: las reas ms altas o ms profundas corresponden a aos con cambios ms acusados.

Aunque existen varios aos con descensos, que determina si la zona est perdiendo poblacin de forma estructural es la lnea de tendencia la que nos va a determinar si la tendencia general es perder, ganar o mantener la poblacion. Si el valor final es inferior al inicial, podemos ver que la tendencia es negativa y que la tendencia es de perder poblacin en esta regin de Navara.
